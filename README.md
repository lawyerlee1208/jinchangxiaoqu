# 金昌市各小区所属街道社区查询系统

## 项目简介

本项目是一个简单而实用的网页应用，旨在帮助金昌市居民快速查询各小区所属的街道和社区信息。通过这个系统，用户可以轻松获取准确的地址信息，方便日常生活中的各种需求。

## 功能特点

- 快速查询：用户只需输入小区名称的一部分，系统就会实时显示匹配结果。
- 详细信息：显示小区名称、所属街道、所属社区等详细信息。
- 完整地址：自动生成包含市、区、街道、社区、小区的完整地址。
- 一键复制：提供复制完整地址的功能，方便用户使用。
- 响应式设计：适配各种设备，包括桌面电脑、平板和手机。

## 使用说明

1. 打开网页后，您会看到一个搜索框。
2. 在搜索框中输入您想查询的小区名称（可以是部分名称）。
3. 系统会实时显示匹配的结果。
4. 在结果中，您可以看到小区的详细信息，包括所属街道和社区。
5. 如果需要使用完整地址，点击"复制地址"按钮即可将地址复制到剪贴板。

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)

## 本地运行

1. 克隆本仓库到本地机器。
2. 使用现代浏览器（如 Chrome、Firefox、Safari 或 Edge）打开 `index.html` 文件。
3. 无需额外的设置或依赖，网页应该可以直接运行。

本项目中的数据基于金昌市官方提供的小区、街道和社区信息。如果您发现任何数据错误或需要更新，请提交 issue 或 pull request。
